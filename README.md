# 🔴 NIRVANA X - Pokédex Focus Dashboard

A gamified, Pokémon-themed productivity and focus-tracking dashboard designed for students and developers. Turn your daily study routine into a regional Pokémon journey!

## ✨ Features
* **Trainer Card UI:** A personalized dashboard header displaying trainer stats and badges.
* **Interactive Pokédex Modules:** * 🗺️ **Schedule:** Plan your deep focus zones and daily routing.
  * 📈 **Progress:** Track XP, streaks, and quest-style milestones.
  * 📊 **Metrics:** Visual HP/XP style bars representing daily focus energy.
  * ⚡ **Distractions:** A battle-log of wild distractions encountered during study sessions.
* **Sliding Side Panel:** A sleek, animated right-side overlay that dynamically loads content based on the selected module.
* **📷 Live Focus Recorder (Prototype):** An integrated webcam module that requests camera permissions and simulates real-time focus tracking (Focused vs. Distracted state logging).

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, grid/flexbox layouts, CSS gradients, animations, and a responsive design (No external CSS framework required).
* **Vanilla JavaScript:** DOM manipulation, dynamic content rendering, and WebRTC integration (`getUserMedia` API) for the camera functionality.
